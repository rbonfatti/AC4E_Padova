# Codex App Track (Lane A2)

Same project files as **Codex CLI**: `.agents/skills` for skills and `.codex/`
for subagents, hooks, and config examples. Differences are UI and resume-path
only.

## Setup

1. Open Codex desktop app → add project folder (your `starter_article` copy).
2. Merge harness from [`../codex/README.md`](../codex/README.md).
3. Review and trust hooks via `/hooks` in the thread.

## Surface notes

- Slash commands and app commands can differ by release; verify in the command
  menu and official app docs.
- Session resume uses app threads, not `codex resume` from a different terminal session.
